## 技术栈

### 后端技术栈

- **框架**: FastAPI (Python 3.10+)
- **向量数据库**: Chroma&#x20;
- **Embedding 模型**: 本地模型LLM API  Embeddings
- **LLM 服务**: 本地模型 API
- **文档处理**: PyPDF2 / python-docx / Unstructured
- **数据验证**: Pydantic
- **异步支持**: asyncio / aiofiles
- **CORS 支持**: fastapi.middleware.cors

### 前端技术栈

- **框架**: React 18 + TypeScript
- **状态管理**: Zustand / Redux Toolkit
- **UI 组件库**: shadcn/ui
- **HTTP 客户端**: Axios&#x20;
- **代码编辑器**: React Monaco Editor / CodeMirror
- **Markdown 渲染**: React Markdown / marked
- **表单处理**: React Hook Form
- **构建工具**: Vite / Create React App

## 后端接口

### API 端点结构

```
/api/v1
├── /chat                    # 问答接口
│   ├── POST /query          # 提交问题

│   └── DELETE /history      # 清空历史

├── /knowledge-base          # 知识库管理
│   ├── GET /                # 获取知识库列表
│   ├── POST /               # 创建知识库
│   ├── GET /{kb_id}         # 获取知识库详情
│   ├── DELETE /{kb_id}      # 删除知识库
│   └── POST /{kb_id}/upload # 上传文档
│
├── /documents               # 文档管理
│   ├── GET /                # 获取文档列表
│   ├── GET /{doc_id}        # 获取文档详情
│   └── DELETE /{doc_id}     # 删除文档
│
├── /embedding               # Embedding 处理
│   ├── POST /encode         # 文本向量化
│   └── GET /models          # 获取可用模型
│
└── /health                  # 健康检查
    └── GET /                # 系统健康状态
```

### 请求/响应示例

#### 1. 提交问题

```bash
POST /api/v1/chat/query
```

**请求体**:

```json
{
  "question": "什么是 RAG?",
  "knowledge_base_id": "kb_001",
  "history": [
    {"role": "user", "content": "你好"},
    {"role": "assistant", "content": "你好！有什么可以帮你的？"}
  ]
}
```

**响应**:

```json
{
  "success": true,
  "data": {
    "answer": "RAG (Retrieval-Augmented Generation) 是一种结合检索和生成的 AI 技术...",
    "sources": [
      {
        "document_id": "doc_001",
        "content": "RAG 是检索增强生成...",
        "score": 0.95
      }
    ],
    "response_time_ms": 1250
  }
}
```

#### 2. 获取对话历史

```bash
GET /api/v1/chat/history?kb_id=kb_001
```

**响应**:

```json
{
  "success": true,
  "data": {
    "messages": [
      {"role": "user", "content": "什么是 RAG?"},
      {"role": "assistant", "content": "RAG 是检索增强生成技术..."}
    ],
    "total": 2
  }
}
```

#### 3. 上传文档

```bash
POST /api/v1/knowledge-base/kb_001/upload
Content-Type: multipart/form-data
```

**请求体**:

```
file: [Binary PDF File]
```

**响应**:

```json
{
  "success": true,
  "data": {
    "document_id": "doc_001",
    "filename": "document.pdf",
    "pages": 15,
    "status": "processing"
  }
}
```

### 数据模型 (Pydantic)

```python
class QuestionRequest(BaseModel):
    question: str
    knowledge_base_id: str
    history: List[Message] = []

class AnswerResponse(BaseModel):
    answer: str
    sources: List[SourceDocument]
    response_time_ms: int

class DocumentUploadResponse(BaseModel):
    document_id: str
    filename: str
    pages: int
    status: str
```
